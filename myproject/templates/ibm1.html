IBM Watson

IBM Watson supports a visual recognition API, which is used to understand content of an image. We can perform image processing using that.

Image processing is defined by analysis of a specified image or to extract some useful information from it. For each image, the response includes a score for each class within each selected classifier. Scores range from 0 - 1 with a higher score indicating greater likelihood of the class being depicted in the image. The default threshold for reporting scores from a classifier is 0.5.

Further, we can also train our custom classifiers by training with appropriate images for a particular class.
